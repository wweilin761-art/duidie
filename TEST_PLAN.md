## 十、发现的问题及修复

### P0 — 影响核心玩法
1. ✅ **资源节点无冷却** — 采集后树/岩石/浆果丛不显示再生计时器
   → 修复: RecipeEngine 对非消耗输入卡自动添加 timerTemplate 冷却
   
2. ✅ **吃掉食物多出村民** — eat_* 配方的 output: villager 会生成新卡
   → 修复: eat_ 配方跳过输出卡生成

3. ✅ **拖拽后卡牌无法再选** — 空间索引未更新
   → 修复: onPointerUp 后调用 updateSpatialIndex + dirty=true

4. ✅ **配方完成后卡牌锁死** — locked 状态未同步到 DOM
   → 修复: syncDOM 始终检查 locked/timer/starving 状态

### P1 — 配方数据问题
5. ⚠️ **cook_meat 配方** — 输入只有 villager，无生肉，从空气产熟肉
6. ⚠️ **eat_fish_cooked** — 与 eat_cooked_meat 重复
7. ⚠️ **奶酪/牛奶恢复量** — restoreVillagerHunger 中未定义,默认为15

### P2 — 功能缺失
8. [ ] 风暴事件(storm) spawnCards 为空数组,无实际效果
9. [ ] 未实现四季机制(冬夏影响食物消耗)
